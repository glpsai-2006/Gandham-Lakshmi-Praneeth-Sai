## Projects

### CivicWatch - Community Issue Reporting System

A full-stack platform where citizens report local civic problems and administrators track and resolve them efficiently.

**Highlights**
- Citizen portal for submitting geo-tagged issue reports with photos
- Admin dashboard with analytics, status management, and filtering
- Interactive map for visualizing all reported issues in the community
- Role-based access control with JWT authentication
- Real-time status tracking and notifications

**Stack**

`React` `Node.js` `Express` `MongoDB` `TailwindCSS` `JWT` `Multer` `Recharts`

[![View Repository](https://img.shields.io/badge/View%20Repository-181717?style=flat-square&logo=github)](https://github.com/glpsai-2006/CivicWatch)

---

### Online Payment Fraud Detection

A machine learning system that classifies financial transactions as fraudulent or legitimate.

**Highlights**
- Random Forest model trained on real transaction behavior patterns
- Analyzes transaction type, amount, and account balance shifts
- Web interface built with Flask for live predictions
- Returns a fraud probability score per transaction

**Stack**

`Python` `Scikit-Learn` `Random Forest` `Pandas` `NumPy` `Flask` `HTML`

[![View Repository](https://img.shields.io/badge/View%20Repository-181717?style=flat-square&logo=github)](https://github.com/glpsai-2006/Online-Payment-Fraud-Detection)
